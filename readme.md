3D Utils

3D 函数操作集

**Under heavy development**

Repo: [https://github.com/jin5354/v3](https://github.com/jin5354/v3)

Doc: [https://jin5354.github.io/v3/](https://jin5354.github.io/v3/)

## Install

```bash
$ npm i v3js --save
```

## usage

```js
import { Vector3, EulerAngles, Matrix4, Quaternion } from 'v3js'
```

## Utils

- Vector3 三维向量
- EulerAngles 欧拉角
- Quaternion 四元数
- Matrix4 矩阵
